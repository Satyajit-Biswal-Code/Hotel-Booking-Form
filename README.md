**HTML5:-**
>> HTML stands for Hyper Text Markup Language. And the 5 is HTML version.
>> It used to create structure of our websites by using its elements and tags like heading,paragraph,form,title tags.
>> It works together with CSS for styling and JavaScript for interactivity.
>> Web browsers read HTML to display content on the screen.
>> It is the foundation of all websites on the internet.

**TAILWIND CSS:-**
>> Tailwind CSS is the framework of the CSS3.
>> It is used for to make responsive and designing our webpages.
>> It is also called as inline css because it works only inline in out code.
>> Tailwind CSS is a utility-first CSS framework used to build modern and responsive web designs.
>> It provides ready-to-use classes for spacing, colors, typography, and layouts.
>> Tailwind allows faster development without writing custom CSS files.
>> It helps maintain consistent design across a project.
>> Tailwind CSS is highly customizable and works well with modern frameworks.

**JAVASCRIPT:-**
>> JavaScript (JS) is a lightweight, interpreted programming language used to make web pages interactive.
>> It allows developers to handle events like clicks, form submissions, and keyboard input.
>> JS can dynamically update HTML and CSS without reloading the page.
>> It is widely used for client-side validation, animations, and API calls.
>> Today, JavaScript also runs on servers using platforms like Node.js.
